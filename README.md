# Context Engineering Papers

Formal theory for persistent context in LLM systems.

## Papers

### 1. The Root Theorem of Context Engineering
*Maximizing signal-to-token ratio in bounded, lossy channels.*

Two constraints — finite context windows and non-zero degradation — produce every limitation in persistent AI systems. This paper derives the governing principle, proves that homeostatic persistence is a necessary consequence, and shows that retrieval and append-only architectures are special cases of a general framework.
